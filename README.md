📚 StudyVault – Student Notes Organizer

StudyVault is a modern, minimal, and efficient student notes management web app designed to help students organize, categorize, and quickly access their study materials.

It provides a clean UI, subject-based filtering, and powerful note management features — making studying more structured and productive.

🚀 Features
🔍 Smart Search
Quickly search notes by title, subject, or keywords
Instant filtering for faster access
🏷️ Subject Categorization
Notes are grouped into subjects like:
Biology
Chemistry
Mathematics
Physics
History
Helps in structured learning and easy navigation
📌 Pin Important Notes
Pin important notes to keep them at the top
Useful for revision and quick access before exams
📊 Dashboard Overview
Displays:
Total Notes
Total Subjects
Pinned Notes
Gives a quick snapshot of your study data
🎨 Clean & Modern UI
Dark-themed elegant interface
Color-coded notes for better visual distinction
Smooth and distraction-free experience
⏱️ Time Tracking
Shows when notes were created (e.g., "5h ago", "2d ago")
➕ Add Notes Easily
Floating "New Note" button for quick creation
Simple and user-friendly input system
🖼️ UI Overview
🏠 Home Dashboard
Displays all notes
Includes search bar and filters
Shows pinned and regular notes separately
📌 Pinned Section
Highlights important notes like:
Cell Division – Mitosis
Quadratic Formula
